# Changelog

All notable changes to this project will be documented in this file.

## 0.1.1 - 2026-06-01

- Updated GitHub workflows and action runtime to Node 24.
- Changed release automation to create GitHub releases instead of requiring npm credentials.
- Narrowed Dependabot automation to npm updates for reliable first-run checks.
- Normalized package metadata for npm packaging.

## 0.1.0 - 2026-06-01

- Initial CLI for repository health scoring.
- GitHub Action wrapper with score, grade, and pass/fail outputs.
- Markdown and JSON report rendering.
- Tests for scoring and report generation.
